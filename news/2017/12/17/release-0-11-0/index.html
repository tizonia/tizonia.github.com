New Tizonia release v0.11.0

Dec 17, 2017 • juanrubio

Tizonia version 0.11.0 codenamed “Roquetas de Mar” is out!.

129 commits have gone into this release (300 files changed, 14109 insertions(+), 2478 deletions(-)).

The focus has been:

  • various improvements and fixes in YouTube playback,
  • fixed regression introduced in v0.10.0 that broke playback from local files for most audio formats,
  • Bash and Zsh completion files,
  • fixes to support building on Ubuntu 17.10 and 18.04,
  • more preparation work for Chromecast support.

As usual CHANGELOG.md contains the full details.

Credits

Many thanks to the following contributors for their pull requests in this release:

  • djsiddz2, pull request #415.

And thanks as well to those who have provided feedback via the project’s bug tracker on GitHub.
