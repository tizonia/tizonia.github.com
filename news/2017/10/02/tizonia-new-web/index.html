New Website!

Oct 2, 2017 • juanrubio

The Tizonia Project has from today a new web site!.

SimplyBuilt, the web builder app and hosting shop that has been very kindly (and for free) hosting Tizonia’s website in the last few years is closing down.

So long SimplyBuilt, and thanks for all the fish!

So.. it was necessary to find a new place. Luckily, finding a replacement has not been difficult. GitHub Pages and Jekyll are these days the ‘de-facto’ standard in free hosting for open source projects.

So there your go, a new web site and blog for Tizonia.
