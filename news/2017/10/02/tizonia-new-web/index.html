Tizonia's new web!

Oct 2, 2017 • juanrubio

The Tizonia Project has from today a new web site!.

SimplyBuilt, the web builder app and hosting shop that has been very kindly (and for free) hosting Tizonia’s website in the last few years is closing down.

So it has been necessary to find a new place.

Luckily, finding a replacement has been extremely easy. GitHub Pages and Jekyll are these days the ‘de-facto’ standard in free hosting for open source projects.

So here it is, the new site and blog for Tizonia.

The site is based on the really nice jekyll-doc-theme.
