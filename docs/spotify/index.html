Configuration

NOTE

Spotify playback requires a Premium subscription.

No OAuth support at the moment for Spotify.

To use Spotify, introduce your credentials in Tizonia's config file.

    # If you are configuring Tizonia for the first time,
    # copy the template to your $HOME folder, like this:

    $ mkdir -p $HOME/.config/tizonia
    $ cp /etc/tizonia/tizonia.conf $HOME/.config/tizonia/tizonia.conf

    # ( now edit $HOME/.config/tizonia/tizonia.conf )

See instructions inside the file for more information:

# Spotify configuration
# -------------------------------------------------------------------------
# To avoid passing this information on the command line, uncomment
# and configure accordingly
#
spotify.user     = user
spotify.password = pass
