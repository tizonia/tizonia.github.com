Snap Package

Snaps

Snaps are containerised software packages that can be installed on many Linux distributions. Tizonia's snap packages can be downloaded and installed for amd64, i386, and armhf devices.

Install from the Global snap store

Tizonia on Snapcraft.io Tizonia on uApp Explorer

To install Tizonia on a Snapd-compatible system, use this command:

   $ sudo snap install tizonia # for the latest stable version
