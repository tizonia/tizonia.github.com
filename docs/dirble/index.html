Configuration

To use Dirble, you need a Dirble API Key in Tizonia's config file.

    ( On first use, Tizonia outputs its configuration file, if it is not there yet )

    $ tizonia --help

    ( now edit $HOME/.config/tizonia/tizonia.conf )

See instructions inside the file for more information:

# Dirble configuration
# -------------------------------------------------------------------------
# To avoid passing this information on the command line, uncomment and
# configure your Dirble Api Key here.
#
# To obtain your Dirble API key, go to https://dirble.com/developer/
#
dirble.api_key = xxxxxxxxxxxxxx
