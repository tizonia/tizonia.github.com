Installation

On a Debian-compatible Linux distribution, install Tizonia and all the dependencies by running:

   $ curl -kL https://github.com/tizonia/tizonia-openmax-il/raw/master/tools/install.sh | bash
   # Or its shortened version:
   $ curl -kL https://goo.gl/Vu8qGR | bash

Important

Please have a look at the installation script before running it on your system. The script is hosted on GitHub.
