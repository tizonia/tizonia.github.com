OpenMAX IL Options¶

The section describes a number of options that are more typically used during development. These command-line options allow querying the OpenMAX IL subsystem and learn about the installed OpenMAX IL components and their supported roles.

OPTIONS¶

-L [ --comp-list ]

Enumerate all the OpenMAX IL components in the system.

-R [ --roles-of-comp ] arg

Display the OpenMAX IL roles found in component <arg>.

-C [ --comps-of-role ] arg

Display the OpenMAX IL components that implement role <arg>.
